3 части HTTP-запроса: Query Params, Headers, Body

Когда браузер или fetch() отправляет запрос на сервер — он состоит из трёх важных частей, через которые передаются данные.



Query Params (параметры запроса)

Передаются в URL, после знака ?.
Используются для фильтрации, поиска или передачи мелких данных (например, страница, фильтр, сортировка).


Example:
    GET https://example.com/api/products?category=phones&limit=10



Body (тело запроса)

Передаётся внутри запроса, но не виден в URL.
Используется, когда мы отправляем или изменяем данные — например, создаём пользователя, добавляем товар и т.д.

Example:
    fetch('https://example.com/api/users/', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        name: 'Ali',
        age: 17
    })
    })



Headers (заголовки запроса)

Это технические данные, которые идут вместе с запросом.
Они не видны пользователю, но говорят серверу, как обработать запрос.

Example:
    fetch('https://example.com/api/users/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Token 123456789'
    },
    body: JSON.stringify({ name: 'Ali', age: 17 })
    })


fetch('https://jsonplaceholder.typicode.com/posts/1', {
  method: 'DELETE'
})
  .then(response => {
    if (response.ok) {
      console.log('Post deleted successfully');
    } else {
      console.log('Failed to delete post');
    }
  })
  .catch(error => console.error('Error:', error));



fetch('https://jsonplaceholder.typicode.com/posts/1', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: 1,
    title: 'Updated Title',
    body: 'Updated body content',
    userId: 1
  })
})
  .then(res => res.json())
  .then(data => console.log('PUT response:', data))
  .catch(err => console.error(err));





fetch('https://jsonplaceholder.typicode.com/posts/1', {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: 'Partially Updated Title'
  })
})
  .then(res => res.json())
  .then(data => console.log('PATCH response:', data))
  .catch(err => console.error(err));
